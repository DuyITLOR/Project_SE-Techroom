I. authRouter
1. login, api-post: http://localhost:4000/api/auth, params: { username }
2. authenticate, api-post: http://localhost:4000/api/auth/authenticate, params: { username, password }

II. adminRouter
1. getAccount, api-get: http://localhost:4000/api/admin/account, params: { role }
2. addAccount, api-post: http://localhost:4000/api/admin/account, params: { userID, fullName, birthday, password, role }
3. searchAccount, api-get: http://localhost:4000/api/admin/account/search, params: { user, role }
4. updateAccount, api-put: http://localhost:4000/api/admin/account, params: { userID, fullName, birthday, password, role }
5. deleteAccount, api-delete: http://localhost:4000/api/admin/account, params: { userID }

III. roomRouter
1. getRoom, api-get: http://localhost:4000/api/admin/room, params: {}
2. addRoom, api-post: http://localhost:4000/api/admin/room, params: { roomID, roomName, note }
3. searchRoom, api-get: http://localhost:4000/api/admin/room/search, params: { roomID }
4. updateRoom, api-put: http://localhost:4000/api/admin/room, params: { roomID, roomName, note }
5. deleteRoom, api-delete: http://localhost:4000/api/admin/room, params: { roomID }

III. courseRouter
1. getCourse, api-get: http://localhost:4000/api/admin/course, params: {}
2. addCourse, api-post: http://localhost:4000/api/admin/course, params: { courseID, courseName, courseNumber, syllabus, equipment }
3. searchCourse, api-get: http://localhost:4000/api/admin/course/search, params: { courseID }
4. updateCourse, api-put: http://localhost:4000/api/admin/course, params: { courseID, courseName, courseNumber, syllabus, equipment }
5. deleteCourse, api-delete: http://localhost:4000/api/admin/course, params: { courseID }

IV. classRouter (Lưu ý: userIDs là 1 mảng, ví dụ ["3124234", "34124", ...])
1. getClass, api-get: http://localhost:4000/api/core/class, params: { userid, role }
2. addUser, api-post: http://localhost:4000/api/core/class, params: { classID, userIDs }
3. getClassByRole, api-get: http://localhost:4000/api/admin/class, params: { classID, role }
4. addClass, api-post: http://localhost:4000/api/admin/class, params: { classID, className, lesPerWeek, classPerWeek, startDate, endDate ,courseID, studentIDs, teacherIDs }
5. searchClass, api-get: http://localhost:4000/api/admin/class/search, params: { classID }
6. updateClass, api-put: http://localhost:4000/api/admin/class, params: { ClassID, className, lesPerWeek, classPerWeek, startDate, endDate, courseID, studentIDs, teacherIDs }
7. deleteClass, api-delete: http://localhost:4000/api/admin/class, params: { classID }
8. showClassInfomation, api-get: http://localhost:4000/api/admin/class/info, params: { classID }

V. discussionRouter
1. getDiscussion, api-get: http://localhost:4000/api/admin/discussion, params:{ classID }
2. createPost, api-post: http://localhost:4000/api/admin/discussion, params: { classID,userID ,content, link } 
3. deletePost, api-delete: http://localhost:4000/api/admin/discussion, params: { postID }
4. updatePost, api-put: http://localhost:4000/api/admin/discussion, params: { postID, content, link }
5. downloadFile, api-get: http://localhost:4000/api/admin/discussion/download, params:{ filename }

VI. timetableRouter
all date format: "yyyy-mm-dd", studentIDs, teacherIDs: arrays of userIDs: ["0123", "0124",...]
1. getAllLessonsForTimetable, api-get: http://localhost:4000/api/admin/timetable, params: { weekStartDate } (weekStartDate format: yyyy-mm-dd)
2. getRelatedLessonsForTimetable, api-get: http://localhost:4000/api/core/timetable, params: { userID, weekStartDate } (weekStartDate format: yyyy-mm-dd)
3. getLessonByID, api-get: http://localhost:4000/api/admin/timetable/lessonInfo, params: { lessonID }
4. getClassNotInTimetable, api-get: http://localhost:4000/api/admin/timetable/classNotInTimetable, params: {}
5. addLesson, api-post: http://localhost:4000/api/admin/timetable/add-lesson, params: { classID, date, sessionNumber, roomID }
6. deleteLesson, api-delete: http://localhost:4000/api/admin/timetable/delete-lesson, params: { lessonID }
7. updateLesson: api-put: http://localhost:4000/api/admin/timetable/update-lesson/, params: {lessonID, updatedInfo}
updatedInfo format: {lessonData, studentIDs, teacherIDs}
lessonData format: {classID, date, sessionNumber, roomID}
studentIDs, teacherIDs: arrays of userIDs.
