I. authRouter
1. login, api-post: http://localhost:4000/api/auth, params: { username }
2. authenticate, api-post: http://localhost:4000/api/auth/authenticate, params: { username, password }

II. adminRouter
1. getAccount, api-get: http://localhost:4000/api/admin/account, params: { role }
2. addAccount, api-post: http://localhost:4000/api/admin/account, params: { userID, fullName, birthday, password, role }
3. searchAccount, api-get: http://localhost:4000/api/admin/account/search, params: { userID, role }
4. updateAccount, api-put: http://localhost:4000/api/admin/account, params: { userID, fullName, birthday, password, role }
5. deleteAccount, api-delete: http://localhost:4000/api/admin/account, params: { userID }

III. roomRouter
1. getRoom, api-get: http://localhost:4000/api/admin/room, params: {}
2. addRoom, api-post: http://localhost:4000/api/admin/room, params: { roomID, roomName, note }
3. searchRoom, api-get: http://localhost:4000/api/admin/room/search, params: { roomID }
4. updateRoom, api-put: http://localhost:4000/api/admin/room, params: { roomID, roomName, note }
5. deleteRoom, api-delete: http://localhost:4000/api/admin/room, params: { roomID }

III. courseRouter
1. getCourse, api-get: http://localhost:4000/api/admin/course, params: {}
2. addCourse, api-post: http://localhost:4000/api/admin/course, params: { courseID, courseName, courseNumber, syllabus, equipment }
3. searchCourse, api-get: http://localhost:4000/api/admin/course/search, params: { courseID }
4. updateCourse, api-put: http://localhost:4000/api/admin/course, params: { courseID, courseName, courseNumber, syllabus, equipment }
5. deleteCourse, api-delete: http://localhost:4000/api/admin/course, params: { courseID }

IV. classRouter (Lưu ý: userIDs là 1 mảng, ví dụ ["3124234", "34124", ...])
1. getClass, api-get: http://localhost:4000/api/core/class, params: { userid, role }
2. addUser, api-post: http://localhost:4000/api/core/class, params: { classID, userIDs }
3. getClassByRole, api-get: http://localhost:4000/api/admin/class, params: { role }
4. addClass, api-post: http://localhost:4000/api/admin/class, params: { classID, className, lesPerWeek, classPerWeek, startDate, endDate ,courseID, userIDs }
5. searchClass, api-get: http://localhost:4000/api/admin/class/search, params: { classID }
6. updateClass, api-put: http://localhost:4000/api/admin/class, params: { classID, className, lesPerWeek, classPerWeek, startDate, endDate, courseID }
7. deleteClass, api-delete: http://localhost:4000/api/admin/class, params: { classID }
